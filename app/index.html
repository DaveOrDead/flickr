Sainsbury's Flickr challenge

In order for the memory of selected/favourited photos to be retained on page refresh this site needs to be running on a local server. MAMP or WAMP are ideal for doing so locally.

Or I set it up on a server if you'd rather not have the hassle ;) of installing something locally: http://davidberner.co.uk/flickr/app

You can tab through the photos and hit the enter key as well as clicking to favourite them.

Tested in Firefox 39, Chrome 48, Safari 9, IE 10, 11 (functionally but not layout), Edge
